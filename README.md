### 🚀 Featured Projects

#### [Google Takeout Processor (GTP)](https://github.com/mshablovskyy/gtp)
**Tech:** `Python`
A data organization tool that solves the problem of messy Google Photos data from Takeout archives. It effectively organizes files and successfully restores original creation and modification dates by parsing complex metadata.

#### [Google Calendar Manager (GCM)](https://github.com/mshablovskyy/gcm)
**Tech:** `Python` | `Google Calendar API` | `REST APIs`
An automation application designed to import, update, and manage calendar events from Excel and CSV files directly into Google Calendar, showcasing hands-on experience with external API integration.

#### [Atlassian-backup](https://github.com/mshablovskyy/atlassian-backup)
**Tech:** `Python` | `REST APIs`
A utility script designed to automate the backup process for Atlassian content, currently supporting Confluence environments.

#### [Parking Scraper](https://github.com/mshablovskyy/parking_scraper)
**Tech:** `Python` | `Scrapy`
Web scraper for Krakow city parking lots. Collects real-time availability data and persists it to a local SQLite database with optional Telegram notifications.
